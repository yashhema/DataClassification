## Configuration Options:

### Option 1: Environment Variable (Recommended)
```bash
# Set environment variable before running tests
export CONTENT_EXTRACTOR_TEST_FILES="/path/to/your/test_files"
pytest content_extractor_tests.py

# Or inline
CONTENT_EXTRACTOR_TEST_FILES="/custom/path" pytest content_extractor_tests.py
```

### Option 2: Modify the fixture directly
Uncomment one of the alternative paths in the fixture:
```python
# For relative to test file location:
test_dir = Path(__file__).parent / "test_files"

# For absolute path:
test_dir = "/absolute/path/to/test_files"
```

### Option 3: pytest configuration file
Create `pytest.ini` or `conftest.py`:
```ini
# pytest.ini
[tool:pytest]
env = 
    CONTENT_EXTRACTOR_TEST_FILES = ./my_test_files
```

### Option 4: Command line parameter
Add this to your test class:
```python
def pytest_addoption(parser):
    parser.addoption(
        "--test-files-dir",
        action="store",
        default="test_files",
        help="Directory containing test files"
    )

@pytest.fixture
def test_files_dir(request):
    return Path(request.config.getoption("--test-files-dir"))
```

Then run: `pytest --test-files-dir="/my/custom/path"`
